# online-courses

Задача: Проектирование системы для онлайн-курсов
Описание задачи:
Проектирование платформы для онлайн-обучения, аналогичной Coursera или Udemy, которая позволяет пользователям создавать, публиковать и проходить курсы. Система должна поддерживать множество пользователей.

Нужно написать user story (штук 10-15) и use case (шт 2-3)

Клиенты: преподаватели, ученики, техническая поддержка.
